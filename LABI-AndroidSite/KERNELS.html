Kernel

O sistema android tem como base o kernel do Linux, mais precisamente do ramo LTS (suporte ao longo do tempo), mas não se trata exatamente do mesmo kernel usado por outros sistemas operacionais baseados em Linux. O kernel Linux usado no android possui profundas alterações na sua arquitetura implementadas pela Google. O kernel é um dos mais importantes componentes de um sistema operativo, sendo o responsável pela interação entre o software e o hardware, ou seja, é o kernel que controla os recursos de um sistema permitindo que os programas façam uso deles. É esta a razão para a qual os desenvolvedores android demoram algum tempo a realizar uma nova port para os seus dispositivos, pois o hardware difere de dispositivo para dispositivo. Segundo alguns engenheiros o android não é Linux no tradicional estilo Unix, pois não inclui certos componentes que são tipicamente encontrados nas distribuições Linux, mas mesmo assim o android é considerada uma distribuição Linux de acordo com a Linux Foundation.

Custom Kernels

Apesar de o Kernel já vir dentro do firmware, este pode ser também desenvolvido à parte. Um Custom Kernel pode ser desenvolvido e optimizado de modo a melhorar a performance, estabilidade, fluidez e bateria do sistema Android. Estas melhorias são obtidas através da modificação do Governador da CPU, do escalonamento e através do overclocking (opcional). O Governador da CPU não é nada mais que o comportamento (perfil) da CPU ao realizar determinadas tarefas. As OEMs, normalmente não permitem ao utilizador escolher o perfil da CPU, por questões de segurança em relação aos utilizadores menos experientes, no entanto qualquer desenvolvedor pode adicionar diferentes perfis de governador da CPU ao seu Custom Kernel. Muito importante na criação de Kernels personalizados é a utilização de Schedulers (escalonadores de processos) editados, já que são eles que vão ditar como o sistema deve funcionar ao alterar os diversos perfils do governador e se quando deve alterar os perfils automanticamente em determinadas situações.
